인공지능의 선구자 들은
머신 러닝은 데이터를 이해하는 알고리즘이자 하나의 어플리케이션 이다. 

20세기 후반 
데이터에서 지식을 추출하여 가까운 미래를 예측 하는 
셀프 러닝 알고리즘과 관련된 인공지능 하위 분야 머신 러닝이 나왔다 

인간이 직접 대량의 데이터를 분석하는 기호주의 방식이 아니라
머신러닝을 통해 더 효율적으로 데이터를 추출하는 알고리즘이 나왔다 

-> 컴퓨터 과학과도 영향을 미치고 우리 일상 생활에서도 영향을 미친다 

지도학습
레이블된 훈련 데이터를 활용하여 모델을 학습 시킨다음 
본적없는 가까운 미래 데이터에 대해 예측값을 출력 하는것 이다
(광고성 이메일 필터링) 
학습이라는 과정을 통해 데이터를 학습 할수 있다. 
모든 클래스 레이블을 새로운 샘플에 할당 할수 있어야 한다. = 손글씨 인식 

분류 vs 회귀
분류 는 과거의 관측을 통해 새로운 샘플의 범주형 클래스 레이블 예측 
이산적 / 순서 가 없음  = 이메일 광고성 인지 감지하는 알고리즘 이 분류에 해당 한다. 
일련의 규칙을 학습 을 한다. 

회귀 
데이터가 주어졌을때 연속적인 출력값을 예측하는 방법
머신러닝 알고리즘은 입력 데이터와 출력 값이 주어졌을때 
두 변수 사이의 관계를 탐색하는 방식 
(프로게이머 지망생에 랭크 점수 ) 
선형 회귀 
입력값 x와 출력값  y 가 주어지면 각데이터 와 직선 사이거리가 최소가 되는 
직선을 그을수 있는데 평균 제곱 거리를 사용한다. 
y = ax + b 값으로 측정 하는데 
머신러닝은 새로운 데이터를 출력 할수가 있다 . 


비지도 학습
레이블 되지 않은 데이터 구조를 알수 없는 데이터 를 다룬다 
의미있는 정보를 추출 하고 데이터 구조를 탐색 

군집 을 많이 사용한다 
군집:사전정보 없이 쌓여있는 그룹 정보를 
의미있는 서브 그룹 또는 클러스터로 조직하는 탐색적 데이터 분석 기법입니다. 
분석과정에서 만든 클러스터는 어느정도 유사성을 공유 하는 관계 이다. 

차원축소 : 고차원의 데이터를 저차원으로 축소하는 기법이다. 
데이터를 일부로 축소하면 학습의 성능이 떨어질수 있다 따라서 

대부분의 정보를 유지하면서 더 작은 차원의 부분공간으로 데이터를 압축 시키게 한다. 

 


강화 학습
환경과 상호작용하여 에이전트의 성능을 향상시키는 분야 
즉 환경에 따라 데이터가 달라진다 
보상 함수를 통해 에이전트가 하는 행동이 좋은 활동인지 별로인지 학습하는 방식 

목표: 보상이 최대가 되는 일련의 행동을 학습을 하는것이다. 
밤낮없이 진행 가능 단기간의 놀라운 성장이 나온다
