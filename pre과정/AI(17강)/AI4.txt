딥러닝에 대해서 
딥러닝을 정의 할때는 머신러닝과 의 차이점을 생각해 보려면
머신러닝의 역할은 
입력 데이터와 예상 출력 셋의 관계를 학습 하느것이 역할이다 . 
머신러닝을 하기 위해서는 3가지 데이터가 필요 
입력 데이터(ex)사람의 음성인식,이미지 데이터 )
예상 출력값 (원고 형태의 데이터 , 강아지 태그 데이터)
알고리즘 정확도 측정법(현재출력 과 예상 출력에 대해서 차이를 측정하면서 학습을 수행한다.)
이 필요 하다 
즉 따라서 머신러닝과 딥러닝의 핵심 과제는  데이터를 의미있는것으로 변환 
예상하는 결과에 더 가깝게 접근할수 있도록 하는 표현 학습 하는 과정이다... 


머신러닝 모델이란
입력 데이터에 대한 적절한 표현을 찾는일 
즉 분류 작업과 같이 현재 직면하고 있는 작업에 더 적합한 데이터 변환 방식을 찾는일과 
관련 되어있다 

구체화 해보면 
x,y 좌표계에서 흰색점과 검은색 좌표가 있다고 하자 
이때 어떤 좌표가 정해지면 이값이 검정인지 흰색인지 예측하는 프로그램을
만든다고 하면 필요한 과정은 3가지 이다 
1. 입력 값은 좌표 이고 
2. 결과 값은 점들의 색이다 
3. 알고리즘이 올바른 작업을 수행하는지 여부를 측정 하려면 올바르게 분류된 점들의 백분율이다.

학습이란 
더나은 데이터로 찾아가는 일  

딥러닝 
딥러닝은 머신러닝의 특정 하위 분야로 
표현들을 점점 더 의미있게 만들어가는 연속 계층들을 학습하게 하는데 중점을 두고 데이터 표현을 학습하는 새로운 방법 

입력층 -> 은닉층 -> 출력층 
딥이라는 문구는 연속된 표현된 층을 나타내는 문자이다. 

계층적 표현을 신경망이라고 하는  모델을 통해 학습되며 
은닉층의 각 계층별 특징 : 교차/반복/결합 등 다양하다 .

신경망은
신경 생물학에서 차용
뇌를 이해한 데서 얻은 영감을 통해 부분적으로 개발 한다.

신경망을 설명하려면 뉴런을 설명 해야 한다 
뉴런 = 신경 세포 라고 하는게 3가지 특징이 있다. 
 
1.뉴런은 네트워크를 형성 한다 
2. 여러 뉴런에 전달되는 신호의 합이 일정 크기 을 넘지 않으면 뉴런은 전혀 반응하지 않습니다. 
3.여러 뉴런에서 전달 되는 신호의 합이 일정 크기 임곗값을 넘으면 뉴런이 반응하며 
다른 뉴런에 일정 강도의 신호를 전달합니다. 
4. 두번쨰 와 세번쨰 과정으로 전달되는 여러 뉴런에 전달되는 신호의 합은 신호마다 가중치가 다르다 

일정 단위의 인위적인 네트워크로 표현한것이 신경망이라고 한다. = 수학적 추상화 라고 한다. 

즉 신경 세포(뉴런의) 집합체를 수학적 모델로 만든게 신경망이고 이런한 계층을 연속되게 구조한것이 딥러닝이라 할수 있다. 

딥러닝의 학습 

입력 x -> 계층 데이터 변환 + 가중치  -> 다수의 계층(데이터 변환)+가중치 -> 예측 y
신경망에서 모든 계층의 가중치에 대한 집합을 찾아 
새로운 입력값이 목표한 값에 올바르게 사상 할수 있도록 하는 반복 작업 

우리가 무엇을 해야 하나면 이 가중치에 적절한 값을 찾아야 한다 
즉 가중치를 조절하는 반복작업이 발생한다.

심층 신경망에서는 수천만개의 가중치가 존재 하며 
모든 가중치들을 대상으로 올바른 값을 찾는 일은 쉽지 않은 일이다 
하나의 가중치를 수정하면 다른 모든 가중치에 영향을 끼친다. 

이떄 가중치를 비교하는 방법이 실제 출력 데이터 와 출력층의 차이 값을 측정하며 비교해야 한다. 
= loss function 이라고 하는데 

두차이(거리점수) 를 계산하여 신경망이 얼마나 잘 수행되었는지 계산 하여 이값을 피드백 신호로 사용해 무작위 가중치 값을 조절하는 일을 수행 한다 = 역전파 알고리즘이라고 한다.
